## Additional Notes
Описание структуры ABON1.
Структура содержит следующие поля:
- `fullName` - фамилия и инициалы абонента
- `phoneNumber` - номер телефона
- `connectionDate` - дата подключения телефона (год, месяц, день)
- `tariffPlan` - тарифный план
- `accruedAmount` - начисленная сумма оплаты
- `accountBalance` - сумма на счёту абонента

### Реализованные функции
1. `createAbonents()` - создание массива до 10 записей (ввод с клавиатуры)
2. `sortAbonentsByName()` - сортировка записей по алфавиту
3. `addBonusForOldAbonents()` - добавление 10 руб. абонентам со стажем >10 лет
4. `printDebtors()` - вывод абонентов с отрицательным балансом после вычета
5. `printTariffStatistics()` - статистика по тарифным планам
